# Two Cyan Obtuse Triangles

## Description
This program draws **two cyan-colored obtuse triangles**.  
- Background color is **orange**.    
- The window title shows **my full name**.  
- Press the **first letter of my name** to close the window.

## How to Run
1. Make sure **GLFW** and **GLAD** are installed.  
2. Compile the program.  
3. Run the program and see the two triangles on the screen.
